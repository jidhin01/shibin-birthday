## 🛠️ Tech Stack

*   **Framework:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
*   **Animations:** [Framer Motion](https://www.framer.com/motion/) (Physics-based drags, springs, and transitions)
*   **Icons:** [Lucide React](https://lucide.dev/)
*   **Styling:** Modern Vanilla CSS featuring:
    *   Neo-brutalist UI components
    *   Glitch typography effects
    *   Dynamic floating background layers
    *   Responsive layouts for mobile roasting

## 🚀 Getting Started

### Prerequisites
*   Node.js (Latest LTS recommended)
*   npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
